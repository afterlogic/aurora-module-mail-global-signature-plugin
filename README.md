# aurora-module-mail-global-signature-plugin

With this plugin enabled and configured, all email messages sent out will have a predefined message appended. It can be useful if you'd like to insert advertisement texts or technical information there.

**Signature** is supplied in `data/settings/modules/MailGlobalSignaturePlugin.config.json` file.
